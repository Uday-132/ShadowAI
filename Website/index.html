Shadow AI Features

Everything you need, floating elegantly on top of your work.

AI & Performance

Real-time AI Responses

Powered by sub-second Groq Llama 3 API hooks to deliver swift solutions, explanations, and key data points.

Screen Capture Analysis

Crop and scan any screen region instantly to perform optical character recognition (OCR) and feed coordinates into Scout AI.

Coding Interview Support

Bypass screen assessment traps by overlaying technical assistance on a click-transparent, stream-safe virtual glass pane.

Live Transcription

Listen to system speaker audio or external microphone streams to transcribe speech queries on-the-fly using Whisper.

Unlimited Transcript Storage

Maintain local persistent databases containing records of all past OCR captures, voice inquiries, and AI generated replies.

Security & Stealth

Invisible on Screen Share

Bypasses OBS Studio, Discord screen sharing, Teams, Zoom, screenshots, and recording hooks completely via display composition filters.

Invisible in Dock

Operates without any system taskbar icon, taskbar preview, or active window tray lists. Runs seamlessly in the background.

Invisible in Task Manager

Bypasses process tracking hooks by running under a masqueraded system name (`SystemCoreHost.exe`) that blends in with standard OS processes.

Invisible to Tab Switching

Excluded from the Windows Alt+Tab dialog stack, preventing accidental menu overlays and keeping your application listings empty.

Cursor Undetectability

Keeps the cursor free from active window capture highlights or input capturing flags, enabling absolute mouse click-through transparency.

Deployment Guide

Follow these quick configuration steps to boot up your dashboard.

1

Download & Extract

Download the latest Windows release (`setup.exe`). Extract the lightweight Shadow AI files into any local directory of your choice.

C:\Program Files\ShadowAI\
2

Provision API Key

Open the Shadow AI settings panel. Paste your **Groq API Key** to enable Whisper Voice queries and Llama 4 Scout crop OCR scanners.

%APPDATA%\OverlayApp\settings.json
3

Trigger Global Keybinds

Double-click the widget tab bar to toggle drag-locking. Press on your keyboard to instantly switch mouse focus.

Ctrl + Shift + C
